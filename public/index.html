mediasoup v3 examples
  • loopback ... single clinet-server video loopback
    • run server: npm run loopback
    • open with browser: loopback.html
  • broadcast ... publish video from 1 pub client, subscribe video by multiple sub client
    • run server: npm run broadcast
    • publisher clinet with browser: publish.html
    • subscriber clinet with browser: subscribe.html
  • multi-party ... video chat for multi-party
    • run server: npm run multiparty
    • open with browser: multi.html
